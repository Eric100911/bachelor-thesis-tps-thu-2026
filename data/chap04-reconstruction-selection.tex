\section{初级顶点与径迹重建}
\label{sec:pv-track-reco}

在CMS事例重建中，初级顶点（Primary Vertex, PV）重建大致分为顶点寻找和顶点拟合两个步骤。前者将同一次束团对撞产生的带电径迹按其空间位置进行聚类，形成若干顶点候选；后者则对每个候选径迹集合估计其顶点位置及其协方差，并提供施加顶点位置约束后的相关径迹参数和协方差，以及每条径迹在顶点拟合中的权重。通过这一顶点重建过程，对撞顶点的空间位置、相对束流线的偏离以及每条径迹与各顶点的关联得以确定，为后续物理对象重建和事例选择提供了必要的空间和关联信息。

重建程序首先从带电径迹集合中，利用束斑（Beam Spot, BS）信息，筛选出适合进行初级顶点重建的带电径迹。默认的筛选条件列于表~\ref{tab:pv-reco-track-selection}；表中归一化径迹拟合 $\chi^2$ 由 Kalman 滤波得到，反映径迹重建的整体质量。这些要求排除了重建质量差、相对束流线的碰撞参数大、测量不确定度大或径迹位于径迹探测器主要几何接收范围之外的径迹，从而提高了顶点寻找过程的稳定性。

经预选的径迹随后按照其在束流方向的位置进行聚类。用于聚类的 $z$ 坐标为碰撞点（impact point, IP）的 $z$ 坐标 $z_{\mathrm{IP}}$，定义为径迹到束流线最近点的位置。

径迹聚类还需要考虑 $z_{\mathrm{IP}}$ 的不确定度。这一不确定度既来自径迹重建算法给出的本征 $z$ 位置不确定度 $\sigma_{d_z}$，也来自束斑的横向宽度，后者会导致径迹碰撞点位置产生额外的 $z$ 方向不确定度 $\sigma_{z,\mathrm{BS}}$。在默认配置下，径迹聚类算法还通过加入一个本征顶点尺寸项来考虑初级顶点位置的本征尺度。这一本征顶点尺度为：
\begin{equation}
\label{eq:vtx-resolution}
  \sigma_{\mathrm{vtx}} = \SI{0.006}{\centi\meter}.
\end{equation}

由此得到有效的 $z$ 方向位置不确定度 $\sigma_{z,\mathrm{eff}}$：
\begin{equation}
\label{eq:z-resolution}
  \sigma_{z,\mathrm{eff}}^2
  =
  \sigma_{d_z}^2
  + \sigma_{z,\mathrm{BS}}^2
  + \sigma_{\mathrm{vtx}}^2
\end{equation}

在聚类过程中，赋予每条径迹的权重与 $\sigma_{z,\mathrm{eff}}^2$ 成反比。这降低了 $z$ 位置不确定度较大的径迹对聚类结果的影响，提高了聚类过程的稳定性和准确性。

% [PENDING]
径迹聚类采用沿 $z$ 方向的一维确定性退火（deterministic annealing, DA）算法~\cite{roseDeterministicAnnealingApproach1990,roseDeterministicAnnealingClustering1998}。该算法从较高的有效温度出发，允许径迹通过软权重与多个顶点原型同时关联；随后逐步降低温度，使不同 $z$ 位置的顶点候选逐渐分离并收敛~\cite{Chabanat:2005zz}。算法的关键参数（退火因子、终止温度、顶点合并距离、径迹降权尺度等）取自 CMSSW 顶点重建模块的默认配置。这些设置使算法能够在事例堆积环境中将来自不同 $pp$ 对撞的径迹分离为多个顶点候选，同时降低非初级径迹或离群径迹对聚类结果的影响。

% [PENDING]
对每个径迹团，随后采用自适应顶点拟合器（Adaptive Vertex Fitter, AVF）进行三维顶点拟合。AVF 是一种迭代加权拟合方法：在每次迭代中，算法根据每条径迹与当前顶点估计值的兼容性为径迹赋予权重。与顶点位置不兼容的径迹被逐步降权，而与顶点兼容的径迹保持高权重。默认的几何退火参数对应于 $\chi^2$ 截断尺度为 $2.5$。因此，AVF 不需要在拟合前对径迹是否属于某一顶点做出硬性判断，而是在拟合过程中通过软权重抑制离群径迹。拟合完成后，每个顶点候选必须满足有效性要求，且其与束流线在横向平面内的距离须小于 $\SI{1.0}{\centi\meter}$。

% [PENDING]
当事例中存在多个初级顶点候选时，顶点按其关联径迹的横动量平方和进行降序排列。在当前实现中，排序变量并非直接的 $\sum p_{\mathrm{T}}^2$，而是
\begin{equation}
  S_{\mathrm{PV}}
  =
  \sum_{i \in \mathrm{PV}}
  \left[
    \max \left(
      p_{\mathrm{T},i} - \sigma_{p_{\mathrm{T}},i},\,0
    \right)
  \right]^2 ,
  \label{eq:pv-ranking}
\end{equation}
其中 $\sigma_{p_{\mathrm{T}},i}$ 为第 $i$ 条径迹的横动量不确定度。排序后，$S_{\mathrm{PV}}$ 最大的顶点被置于顶点集合的第零个索引位置，通常被视为硬散射过程的主导初级顶点候选。

确定初级顶点后，每条带电径迹需与一个已重建顶点建立关联。关联判断中常用的量为纵向碰撞参数 $d_z$（径迹相对于参考顶点的 $z$ 坐标差）和横向碰撞参数 $d_{xy}$（径迹到顶点在横平面内的最短距离）。主要的关联判据汇总于表~\ref{tab:pv-association}。

初级顶点重建完成后，CMS粒子流（particle-flow, PF）算法重建的候选将与已重建的初级顶点进行关联。关联策略采用表~\ref{tab:pv-association} 所列的层级判据，依物理可靠性从强到弱依次为：顶点拟合隶属关系、瞬发类 $d_z$ 兼容性、重味强子衰变径迹匹配，以及最近 $z$ 距离弱关联。

% [PENDING]
CMS硅径迹探测器中带电粒子的径迹采用基于 Kalman 滤波方法的迭代组合径迹寻找器（Combinatorial Track Finder, CTF）进行重建~\cite{Fruhwirth:1987:Kalman,CMS_TDR_CH6_TRACKING}。重建序列依次包括击中聚类、种子生成、模式识别（径迹寻找）、歧义消除（移除共享过多击中的冗余径迹）和最终径迹拟合。

% [PENDING]
原始信号首先在像素和条状传感器中进行聚类。在像素探测器中，相邻像素间的电荷共享提供了二维位置信息，在偏转平面内的空间分辨率约为 10~$\mu$m。在条状探测器中，相邻条上超过阈值的信号被合并为一维击中团。聚类完成后，径迹重建从种子生成开始：利用两个或三个像素击中以及第一个条状层的击中组合，估计初始的五维螺旋线参数及其协方差矩阵，为后续 Kalman 滤波提供初值。

% [PENDING]
CTF 将每个种子向外逐层传播穿过后续径迹探测器层，利用 Kalman 滤波寻找与径迹候选兼容的击中，并在每一层更新径迹参数。当一个候选可以通过多种击中分配方式传播时，算法并行维护多个径迹候选，并在每一层根据与分配击中的 $\chi^2$ 兼容性进行剪枝。遍历所有径迹探测器层后，从外向内应用第二次 Kalman 滤波（平滑器），以获得每个测量面上的最优参数估计。

% [PENDING]
为提高不同径迹拓扑的重建效率，CTF 以多次迭代方式运行。每次迭代后，明确分配给高质量径迹的击中将从后续搜索中移除，以减少组合复杂度。在 Run~1 数据处理中，CMS 采用六次迭代，分别针对特定的径迹拓扑，如高 $p_{\mathrm{T}}$ 瞬发径迹、低 $p_{\mathrm{T}}$ 径迹以及来自长寿命粒子衰变的位移径迹。

% [PENDING]
若数条重建径迹共享了大部分击中，歧义消除步骤将按击中数和 $\chi^2$ 对径迹排序，保留最优径迹，并移除冗余候选。最终重建径迹按照击中数、$\chi^2$ 以及与初级顶点的兼容性分为若干质量类别（loose、tight、high-purity）；其中 high-purity 类别用于物理分析。对横动量 $p_{\mathrm{T}} > \SI{1}{\giga\eV}$ 的带电粒子，径迹重建效率约为 95\%，假径迹率低于百分之几，中心区域在 $p_{\mathrm{T}} \sim \SI{10}{\giga\eV}$ 处的横动量分辨率约为 1--2\%~\cite{CMS_TDR_CH6_TRACKING}。

\section{缪子重建与选择}
\label{sec:muon-reco-selection}

% [PENDING]
CMS 中的缪子候选通过联合硅径迹探测器和缪子系统的信息进行重建；缪子探测器的硬件结构和几何覆盖已在第~\ref{chap:experiment-dataset}~章介绍。离线重建中，standalone 缪子径迹首先由缪子系统中的局部径迹段构建，并在考虑能量损失、多次散射和磁场非均匀性的条件下向内外传播。global 缪子重建则将 standalone 缪子径迹与硅径迹探测器中的 tracker 径迹进行匹配，并利用两个探测系统的击中进行联合拟合。由于综合利用了内外两个探测系统的信息，global 缪子重建在横动量分辨率上通常优于单独使用缪子系统的重建~\cite{CMS_TDR_CH9_MUONS,CMS_MUON_PERF_13TEV}。

% [PENDING]
在线触发和离线重建都依赖缪子系统与硅径迹探测器之间的一致性匹配。对本分析而言，低横动量缪子的重建效率和双缪子质量分辨率直接影响 $J/\psi$ 候选的接受度；因此后续选择采用针对低动量缪子优化的鉴别工作点，并通过触发效率修正吸收在线筛选带来的事例级效率影响。

% [PENDING]
本分析采用 CMS soft muon 工作点作为缪子鉴别判据~\cite{CMS_MUON_PERF_13TEV}。该工作点针对低横动量缪子进行了优化，特别适用于 $B$ 物理和夸克偶素相关的末态。在探测器层面，soft muon 工作点由以下要求所定义：

\begin{enumerate}
  \item 候选必须被重建为 tracker 缪子，且其内层 tracker 径迹需满足 high-purity 质量要求。
  \item tracker 径迹必须在至少六层硅径迹探测器层中具有击中，其中至少包含一个像素探测器击中；
  \item tracker 缪子的径迹段匹配必须为严格匹配：外推 tracker 径迹与匹配到的缪子段在局部 $x$ 和局部 $y$ 两个坐标方向上的 pull 值均须小于 3
  \item 候选必须与初级顶点松散兼容，横向和纵向碰撞参数需分别满足 $|d_{xy}| < \SI{0.3}{\centi\meter}$ 和 $|d_z| < \SI{20}{\centi\meter}$。
\end{enumerate}

soft muon 鉴别策略的核心思想是主要依赖内层硅径迹探测器提供的高精度动量和碰撞参数测量，同时用缪子系统中的径迹段匹配来确认缪子假设。这在低横动量区域实现了高缪子鉴别效率。

在 soft muon 鉴别的基础上，本分析对缪子施加如下运动学选择条件：

\begin{itemize}
  \item 赝快度：$|\eta| < 2.4$，确保缪子位于 CMS 硅径迹探测器和缪子谱仪的有效几何接收范围内。
  \item 横动量：根据缪子探测器触发接受度和重建分辨率随 $\eta$ 的变化，采用分区阈值：
    \begin{itemize}
      \item 桶部区域（$|\eta| < 1.2$）：$p_{\mathrm{T}} > \SI{3.5}{\giga\eV}$；
      \item 端盖区域（$1.2 \leq |\eta| < 2.4$）：$p_{\mathrm{T}} > \SI{2.5}{\giga\eV}$。
    \end{itemize}
\end{itemize}
这一分区 $p_{\mathrm{T}}$ 阈值设计兼顾了触发效率和分析纯度的需求。

\section{\texorpdfstring{$J/\psi$}{J/psi} 候选重建}
\label{sec:jpsi-reco}


$J/\psi$ 介子通过其双缪子衰变道 $J/\psi \to \mu^+\mu^-$ 进行重建。该衰变道的世界平均分支比和不确定度已在第~\ref{sec:theory-quarkonium-production}~节给出。尽管双缪子分支比较小，但末态中的两个缪子提供了清晰的实验特征。借助 CMS 优异的缪子重建和鉴别能力，此衰变道可提供高信噪比，并便于在复杂本底环境中进行信号提取。

$J/\psi$ 候选由满足第~\ref{sec:muon-reco-selection}~节所述选择条件的缪子中选取带相反电荷的缪子对进行重建。每个候选需满足以下判据：

\begin{enumerate}
  \item 不变质量窗口：双缪子不变质量 $m_{\mu\mu}$ 需满足 $2.9 < m_{\mu\mu} < \SI{3.3}{\giga\eV}$；
  \item 横动量和快度：$J/\psi$ 候选的横动量需满足 $p_{\mathrm{T}}^{J/\psi} > \SI{6.0}{\giga\eV}$，快度需满足 $|y^{J/\psi}| < 2.5$；
  \item 顶点拟合质量：两条缪子径迹必须拟合到公共衰变顶点，且顶点拟合概率须大于 $1\%$。
\end{enumerate}

% [PENDING]
在上述选择条件中，不变质量窗口兼顾了 $J/\psi$ 的质量峰分辨率与连续本底的抑制。横动量和快度选择基于 CMS 的触发效率和重建性能。顶点拟合质量要求有助于抑制组合本底，并确保两条缪子在空间上与源自同一衰变顶点一致。

\section{\texorpdfstring{$\phi$}{phi} 候选重建}
\label{sec:phi-reco}

% [PENDING]
$\phi$ 介子通过其主要衰变道 $\phi \to K^+K^-$ 进行重建，其世界平均分支比和不确定度见第~\ref{sec:theory-quarkonium-production}~节。在 CMS 实验中，对于横动量在 \si{\giga\eV} 量级的带电强子，区分 $K^\pm$ 与 $\pi^\pm$ 的能力有限。因此，本分析采用 CMS 多夸克偶素分析中常用的策略：所有通过基本质量选择且未被鉴别为缪子的带电径迹均赋予 $K^\pm$ 质量假设~\cite{CMS_TRI_JPSI}。

% [PENDING]
$\phi$ 候选由满足以下条件的带电径迹中选取带相反电荷的径迹对进行重建：

\begin{enumerate}
  \item 径迹质量：带电径迹需满足 CMS high-purity 质量要求，以保证可靠的径迹重建。
  \item 运动学接收：径迹赝快度需满足 $|\eta| < 2.5$，横动量需满足 $p_{\mathrm{T}} > \SI{2.0}{\giga\eV}$。
  \item 不变质量窗口：在 $K^\pm$ 质量假设下，双径迹不变质量 $m_{KK}$ 需满足 $\SI{0.99}{\giga\eV} < m_{KK} < \SI{1.07}{\giga\eV}$，对应于以 PDG 给出的 $\phi$ 质量 $1019.5~\mathrm{MeV}$ 为中心、宽度约为 $\pm 40~\mathrm{MeV}$ 的窗口~\cite{PDG2024}。
  \item 横动量：$\phi$ 候选的横动量需满足 $p_{\mathrm{T}}^{\phi} > \SI{4.0}{\giga\eV}$。该阈值的选择基于以下物理考虑：$\phi(s\bar{s})$ 为轻味介子，在质子--质子对撞中通过软 QCD 过程大量产生，主要集中在 $p_{\mathrm{T}} \lesssim \SI{1}{\giga\eV}$ 区域。施加较高的 $p_{\mathrm{T}}$ 阈值可以有效抑制来自软 QCD 过程的组合本底，并有助于确保重建的 $\phi$ 候选源自硬散射过程。
  \item 顶点拟合质量：两条带电径迹必须拟合到公共顶点，且顶点拟合概率须大于 $1\%$，以抑制来自随机径迹对的组合本底。
\end{enumerate}

上述选择策略的核心思路是通过$p_T$阈值筛选硬散射产生的$\phi$介子，同时以径迹质量、不变质量窗口和顶点拟合要求抑制组合本底。与$J/\psi$候选的选择（第\ref{sec:jpsi-reco}节）相比，$\phi$候选的挑选更侧重于压低来自软QCD过程的贡献。以下第\ref{sec:triplet-reco}节将$J/\psi$和$\phi$候选组合为事例级的$J/\psi J/\psi\phi$候选。

\section{\texorpdfstring{$J/\psi J/\psi\phi$}{J/psi J/psi phi} 候选重建}
\label{sec:triplet-reco}

在重建并筛选出单个末态粒子，即缪子、$J/\psi$ 候选和 $\phi$ 候选之后，本分析构建事例级的 $J/\psi + J/\psi + \phi$ 候选组合：

\begin{enumerate}
  \item 从事例中选取两个满足第~\ref{sec:jpsi-reco}~节所述条件的 $J/\psi$ 候选，要求它们不共享任何缪子径迹；
  \item 从事例中选取一个满足第~\ref{sec:phi-reco}~节所述条件的 $\phi$ 候选，要求其两条带电径迹不与两个 $J/\psi$ 候选使用的缪子径迹重叠；
  \item 将两个 $J/\psi$ 候选拟合到公共顶点，要求顶点拟合概率大于 $1\%$，以确保两个 $J/\psi$ 候选在空间上与源自同一对撞顶点一致~\cite{CMS_TRI_JPSI}；
  \item 将两个 $J/\psi$ 候选和 $\phi$ 候选联合拟合到同一顶点，并要求拟合成功并给出有效顶点，以进一步加强三个夸克偶素同时产生的信号假设。
\end{enumerate}

若一个事例中有多个候选组合满足上述条件，则选取$J/\psi+J/\psi+\phi$之横动量平方和最大的组合。
这一选择是出于排除组合本底的考虑：在随机组合的背景下，较低的横动量一般与软QCD过程或其他本底的贡献相关，而信号事件更可能具有较高的横动量。通过选择横动量平方和最大的组合，可以在保持信号效率的同时有效抑制来自随机组合的本底。

上述层级化的筛选策略体现了从单个物理对象到完整事例的逻辑递进。首先通过 soft muon 工作点和运动学选择确保单个缪子的质量；随后利用双缪子不变质量和顶点拟合鉴别 $J/\psi$ 候选，并以类似方式构造 $\phi$ 候选；最后在事例层面通过多介子公共顶点约束来检验三个夸克偶素的同时产生。这一设计逐步抑制了来自随机组合、连续过程和软 QCD 过程的本底贡献，同时尽可能保持信号效率。
