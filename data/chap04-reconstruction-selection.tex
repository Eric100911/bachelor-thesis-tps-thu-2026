\chapter{事例重建与选择策略}

\section{\texorpdfstring{缪子与 \(J/\psi\) 候选重建}{缪子与 J/psi 候选重建}}

\(J/\psi + J/\psi + \phi\) 末态的核心实验特征是两个双缪子候选与一个额外的双径迹候选。
因此，本章首先需要明确缪子对象的选择条件、顶点拟合要求以及 \(J/\psi\) 质量窗口定义。

在最终版本中，这一节应逐项写清：

\begin{itemize}
  \item 单缪子运动学和质量要求；
  \item 双缪子顶点质量要求；
  \item \(J/\psi\) 候选排序规则；
  \item 若存在多重候选，采用何种去重或优选策略。
\end{itemize}

\section{\texorpdfstring{\(\phi\) 候选重建}{phi 候选重建}}

相较于双 \(J/\psi\) 分析，本文特有的部分是 \(\phi\) 候选的引入。
这一节需要给出构成 \(\phi\) 候选的带电径迹选择、粒子假设、质量窗口和顶点约束，
并说明其与组合背景抑制之间的关系。

\section{事件级选择与最终候选定义}

在构建出两个 \(J/\psi\) 候选和一个 \(\phi\) 候选后，需要定义事件级的最终选择：

\begin{itemize}
  \item 候选之间是否允许共享径迹；
  \item 事件中有多个候选组合时采用何种保留规则；
  \item 哪些控制分布用于验证选择的合理性；
  \item 最终信号区与旁带区如何定义。
\end{itemize}

这部分内容应尽量按“对象选择、组合规则、事件判据、控制检查”的顺序撰写，
避免把所有 cut 直接堆叠成列表而缺少物理解释。
