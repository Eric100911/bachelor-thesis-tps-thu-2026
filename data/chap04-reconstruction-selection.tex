\section{初级对撞顶点重建}

\label{sec:pv-reco}
% [TODO] 提供自ChatGPT （GPT-5.5），细节表述还需要修改。


在 CMS 事件重建中，初级顶点（Primary Vertex, PV）的重建大致分为顶点寻找（Vertex Finding）和顶点拟合（Vertex Fitting）两个步骤。前者将同一束团交汇中产生的带电径迹按照其空间位置分组，形成若干顶点候选；后者则在每个候选径迹集合内估计顶点位置与相应协方差，并提供各条径迹在加入顶点位置约束后的相关参数与相应协方差，以及径迹在该顶点拟合中的权重。经上述顶点重建过程，对撞顶点的空间位置、与束流线的相对位置以及每条径迹与各个顶点的关联关系都得到了估计，从而为后续的物理对象重建和事件选择提供了重要的空间信息和关联信息。

\subsection{初级顶点的寻找}

\label{sec:pv-reco_pv-finding}

首先，重建程序从带电径迹集合中，综合束斑信息，选择适合用于初级顶点重建的带电径迹（默认选择条件见表~\ref{tab:pv-reco-track-selection}）。这些条件的物理作用是去除重建质量较差、对束流线碰撞参数过大、测量误差过大，或超出径迹探测器主要几何覆盖范围的径迹，从而提高顶点寻找过程的稳定性。

通过预筛选的径迹随后按照其沿束流方向的位置进行聚类。聚类使用的$z$ 坐标取径迹在相对于束流线的最近接近处，即碰撞点（Impact Point, IP）的 $z$ 坐标 $z_{\mathrm{IP}}$。

径迹聚类时同时需要考察 $z_{\mathrm{IP}}$ 的不确定度。这一不确定度既来自径迹重建算法本身带来的 $z$ 方向位置误差 $\sigma_{d_z}$，亦有束斑在横向平面内宽度对 $z_{\mathrm{IP}}$ 不确定度的贡献：束斑在横向平面内的宽度，赋予径迹的碰撞点位置一项额外的 $z$ 方向不确定度 $\sigma_{z,\mathrm{BS}}$。在默认配置中，径迹聚类算法还会考虑到初级顶点位置的本征尺度，并额外加入一个本征顶点尺度项。在默认配置中，该本征顶点尺度为：

\begin{equation}
  \sigma_{\mathrm{vtx}} = \SI{0.006}{cm}.
\end{equation}

这导出有效的 $z$ 方向位置不确定度 $\sigma_{z,\mathrm{eff}}$：

\begin{equation}
  \sigma_{z,\mathrm{eff}}^2
  =
  \sigma_{d_z}^2
  + \sigma_{z,\mathrm{BS}}^2
  + \sigma_{\mathrm{vtx}}^2
\end{equation}

在聚类过程中，被用于聚类的径迹的权重被取为与 $\sigma_{z,\mathrm{eff}}^2$ 成反比，这有利于降低那些 $z$ 位置不确定度较大的径迹对聚类结果的影响，从而提高聚类的稳定性和准确性。

径迹聚类采用一维 $z$ 方向的确定性退火算法
（deterministic annealing, DA）\cite{roseDeterministicAnnealingApproach1990,roseDeterministicAnnealingClustering1998}。
该算法从较高的有效温度开始，使径迹可以以较软的权重同时关联到多个
顶点原型；随后逐步降低温度，使不同 $z$ 位置的顶点候选逐渐分离并收敛。
CMS 实验中将这一算法应用于顶点寻找的细节见文献~\cite{Chabanat:2005zz}。
默认配置中，退火因子为
$0.6$，聚类时只考虑位于径迹附近约 $4\sigma\sqrt{T}$ 范围内的顶点候选，
退火停止温度为 $T_{\mathrm{stop}}=0.5$，顶点分裂阶段结束温度为
$T_{\min}=2.0$。相距小于 $\SI{0.01}{cm}$ 的中间顶点原型会被合并。
此外，横向碰撞参数显著性较大的径迹会在聚类中被降权；默认降权尺度为
$d_0/\sigma_{d_0}=3$。
\textbf{[待添加引用：以上 DA 顶点寻找的默认参数来源于 CMSSW 源代码
（\texttt{github.com/cms-sw/cmssw}）中的顶点重建模块配置。]}
这些设定使得算法能够在 pileup 环境下根据
$z_{\mathrm{PCA}}$ 将来自不同 pp 相互作用的径迹分成多个顶点候选，同时
降低非初级径迹或离群径迹对聚类结果的影响。

\subsection{初级顶点的拟合}
\label{sec:pv-reco_pv-fitting}

% [TODO] 核查AVF是否确实不需要预先决定径迹的权重。

对每个径迹团簇，随后使用自适应顶点拟合算法
（Adaptive Vertex Fitter, AVF）进行三维顶点拟合。AVF 是一种迭代加权
拟合方法：在每次迭代中，算法根据径迹与当前顶点估计之间的相容性为径迹
赋予权重；与顶点位置不相容的径迹被逐渐降权，而与顶点相容的径迹保持
较高权重。默认几何退火参数对应的 $\chi^2$ 截断尺度为 $2.5$。因此，
AVF 不需要在拟合前硬性决定某条径迹是否属于某个顶点，而是在拟合过程中
以软权重的方式抑制离群径迹。拟合完成后，每个候选顶点需要满足有效性
要求，并且与束流线在横向平面内的距离小于 $\SI{1.0}{cm}$。

当一个事件中存在多个初级顶点候选时，顶点按照其关联径迹的横动量平方和
从大到小排序。当前实现中使用的排序量并非裸的 $\sum p_{\mathrm{T}}^2$，
而是
\begin{equation}
  S_{\mathrm{PV}}
  =
  \sum_{i \in \mathrm{PV}}
  \left[
    \max \left(
      p_{\mathrm{T},i} - \sigma_{p_{\mathrm{T}},i},\,0
    \right)
  \right]^2 ,
  \label{eq:pv-ranking}
\end{equation}

其中 $\sigma_{p_{\mathrm{T}},i}$ 为第 $i$ 条径迹的横动量误差。排序后
$S_{\mathrm{PV}}$ 最大的顶点位于顶点集合的第零个位置，通常被视为
硬散射过程的主顶点候选。

在完成初级顶点重建后，CMS 的粒子流（Particle Flow, PF）算法所重建得到的候选粒子将与重建的初级顶点进行关联。对于每个带电 PF 候选粒子，算法首先检查其带电径迹是否已经在某个顶点拟合中被使用，并且在该顶点中具有非零拟合权重。如果满足这一条件，则该顶点被认为是该粒子最可靠的来源顶点。这是因为在顶点拟合阶段，该径迹已经参与了该顶点位置的估计，并且具有较高的相容性。若一条径迹未能通过顶点拟合权重被明确
归属，则根据表~\ref{tab:pv-association} 中的分层判据继续判断。该分层
结构体现了从强到弱的物理信息：优先使用顶点拟合成员关系，其次使用
prompt-like 的 $d_z$ 相容性，再考虑来自重味强子衰变的 displaced 径迹，
最后才使用最近 $z$ 距离作为弱关联。

\section{缪子的重建与鉴别}
\label{sec:muon-reco-selection}

\subsection{缪子的重建}
\label{sec:muon-reco-selection_reco}

CMS 中的缪子候选体通过综合硅径迹探测器与缪子谱仪的信息进行重建。标准缪子重建流程首先分别在内层硅径迹探测器和外层缪子系统中独立重建径迹。后者称为 standalone muon 重建：它从漂移管（Drift Tune, DT）和阴极条室（Cathode Strip Chamber, CSC）中局部重建得到的径迹段出发，并结合阻性板室（Resistive Plate Chamber, RPC）的击中信息，利用 Kalman 滤波方法构建缪子系统中的径迹。随后，global muon 重建将 standalone muon 径迹与硅径迹探测器中的 tracker track 进行匹配，并使用来自内层径迹探测器和缪子系统的击中信息进行联合拟合。由于同时利用了两套探测器的信息，global muon 重建在高横动量缪子的动量测量中具有较好的分辨率~\cite{CMS_MUON_PERF_13TEV}。

除上述由外向内的重建方式外，CMS 还使用由内向外的 tracker muon 重建算法。该算法从硅径迹探测器中已重建的 tracker track 出发，将满足
$p_{\mathrm{T}} > 0.5~\mathrm{GeV}$ 且 $p > 2.5~\mathrm{GeV}$
的径迹外推到缪子系统。如果外推径迹与至少一个 DT 或 CSC 径迹段相匹配，则该径迹被标记为 tracker muon。匹配在缪子室的局部坐标系中进行，主要使用弯曲平面内测量精度较高的局部 $x$ 坐标；当位置差满足
$|\Delta x| < 3~\mathrm{cm}$
或相应的 pull 小于 4 时，认为 tracker track 与缪子径迹段匹配。tracker muon 对低横动量缪子以及探测器几何覆盖较弱的区域尤为重要，因为这些缪子可能无法形成完整的 standalone 或 global muon。

\subsection{缪子鉴别}
\label{sec:muon-reco-selection_discrimination}

本分析采用 CMS 的 soft muon 工作点作为缪子的基础筛选条件。该工作点针对低横动量缪子优化，尤其适用于 $B$ 物理和夸克偶素相关末态。soft muon 首先要求候选体被重建为 tracker muon，并且其内层 tracker track 满足 high-purity 质量要求。该 tracker track 还必须在至少六层 tracker 中具有击中信息，其中至少包含一层 pixel 击中。为了在保持低横动量缪子效率的同时抑制强子穿透和随机匹配背景，soft muon 工作点要求 tracker muon 的径迹段匹配较为严格：外推 tracker track 与匹配缪子径迹段之间在局部 $x$ 和局部 $y$ 坐标方向上的 pull 均需小于 3。最后，候选体还需与主顶点保持较宽松的一致性，其横向和纵向冲击参数分别满足
$|d_{xy}| < 0.3~\mathrm{cm}$
和
$|d_z| < 20~\mathrm{cm}$。
总而言之，soft muon 选择主要依赖内层径迹探测器提供的精确动量和冲击参数测量，同时利用缪子系统中的径迹段匹配来确认缪子假设~\cite{CMS_MUON_PERF_13TEV}。
